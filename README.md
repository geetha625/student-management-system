# Student Management System using Python

A menu-driven Python application to manage student records with CRUD operations and JSON file handling.

## Features

- Add Student
- View Students
- Search Student by ID
- Update Student Details
- Delete Student
- Find Topper
- Prevent Duplicate Student IDs
- Save Data to JSON
- Load Data Automatically on Startup

## Technologies Used

- Python
- JSON File Handling
- Visual Studio Code
- Git & GitHub
